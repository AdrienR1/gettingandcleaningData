This programs (run_analysis.R) aims at generating 2 outputs with the merge of training and test sets:
1) the measurements on the mean and standard deviation for each measurement. 
2) the average of each variable for each activity and each subject.

The dataset includes the following files:
=========================================

- 'README.txt'

- 'run_analysis.R' : the program. Data set must be in work directory

- 'measurements_info.txt': Shows information about the variables used on the measurements
